
# PyChess - Chess in Python

play chess in python using pygame and pygame_gui.
this project is mainly just intended as an exercise.
currently, there is a chess-board with moveable pieces, without any
rules implemented though. this project is still very much work-in-progress.

# How to Play

just run src/main.py for now. if you have pygame installed, you should see a chess-board show up.
